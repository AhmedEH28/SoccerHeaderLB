# Football Event Labeling 

##  Workflow

1. Open the website.
2. Enter your name in `Annotator`.
3. Click `Open local video`.
4. Select the football video from the computer or USB drive.
5. The app automatically fills the video name and FPS 
6. Watch the video and pause at the moment action happens.
7. Then you will the current pasue time in this format: `hour:minute:second.millisecond`, for example `00:13:20.100`.
8. If the action moment is correct, click `Use Current Time` to copy that time into the timestamp box.
9. If the time is not exact, use `-1 frame`, `+1 frame`, `-1s`, or `+1s` to move backward or forward or watch in slow speed, then click `Use Current Time`.
10. Select the action type.
11. If the action is a header, click `Mark HEADER at Current Time`.
12. If the action is not a header, click `Mark NON-HEADER at Current Time`.
13. You can also add visibility, team, and notes if needed.
14. When finished, click `Export CSV` and 
15. You can also chcek it at review saved rows section

# Note: The header action is the most important one
